# GitHub Upload

Open a terminal in this folder:

```powershell
cd E:\Codex\houdini\Py_untriangulate
```

Initialize and commit:

```powershell
git init
git add .
git commit -m "Add Py_untriangulate Houdini HDA"
```

Create an empty GitHub repository, then connect and push:

```powershell
git remote add origin https://github.com/YOUR_NAME/Py_untriangulate.git
git branch -M main
git push -u origin main
```

Replace `YOUR_NAME` with your GitHub username or organization.
